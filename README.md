# :desktop_computer: 이니스프리 클론사이트 제작 :seedling:

## :hand: 소개

 이니스프리의 웹사이트를 클론하여 만든 포트폴리오 프로젝트로, 이니스프리만이 가진 브랜드 가치를 이해하고 그에 맞는 디자인을 구현할 수 있도록 노력하였습니다. 웹사이트에 사용된 이벤트들을 살펴보며 사용된 코드를 정리하였습니다.
